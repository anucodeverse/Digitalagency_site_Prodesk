# Prompts Used in ProDesk Project

 1. CSS Flexbox Alignment Confusion
      I am confused about the difference between `justify-content` and `align-items` in Flexbox. Can you explain when to use each with real examples from a navbar     layout?

 2. Dark Mode Logic Issue
      I am using `body.classList.contains("dark")` in JavaScript, but I never explicitly added the "dark" class in HTML. How does this logic work, and how should I     properly implement dark mode toggle?

 4. Responsive Navbar Problem
      My navbar works on desktop but not properly on mobile. How can I structure CSS so that links stack or convert into a hamburger menu?

 5. Emoji Not Displaying
      I want to add a moon emoji to my webpage. What is the correct way to include it so it displays properly across browsers?

 6. CSS Property Usage Learning
     I am learning CSS step by step. Instead of designing full pages, can you explain individual CSS properties (like margin, padding, flex, position) with simple      examples and when to use them in real projects?

 7. Toggle Button Functionality
     I am trying to implement a toggle button (for features like dark mode or menu visibility) using JavaScript. How can I properly switch states using                `classList.toggle()` and ensure the UI updates correctly based on the current state?

 8. Converting Articles to Card Layout
    I initially structured my content using semantic `<article>` tags, but I want to redesign them into visually appealing card components. What CSS properties       (like box-shadow, padding, border-radius, flex/grid) should I use to transform articles into responsive cards while maintaining good UI/UX?

 9. Media Query for Responsive Design
    I am building a responsive website and want to adjust layout for mobile, tablet, and desktop. How should I structure media queries in CSS, and what breakpoints    should I use to ensure proper responsiveness without breaking the layout?
